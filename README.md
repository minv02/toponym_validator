# Валидация и обогащение топонимов РФ

Приложение предназначено для:
- валидации мест рождения респондентов по эталонному справочнику населённых пунктов РФ;
- восстановления (обогащения) пропущенных административных уровней;
- расчёта эвристического скора уверенности результата;
- логирования всех проблем и принятых алгоритмом решений.

## Ключевые принципы
1. **Эталонный справочник считается истинным**
2. **Никакого ML** — только воспроизводимые эвристики
3. **Прозрачность** — каждое решение логируется
4. **Fail-soft** — частично заполненные, но согласованные данные обогащаются

## Структура
- `app.py` — Gradio UI
- `core/` — бизнес-логика
- `core/normalization.py` — очистка и стандартизация строк
- `core/matching.py` — нечеткое сопоставление
- `core/enrichment.py` — восстановление данных
- `core/scoring.py` — расчёт уверенности
